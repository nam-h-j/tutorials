# crm-nuxt-tutorial

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

- 사용자가 원하는 폼 서식을 생성해서 사용하도록 하는 컴포넌트 컨셉
